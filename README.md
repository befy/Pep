#### CocoaPods
You can use [CocoaPods](http://cocoapods.org/) to install `Pep` by adding it to your `Podfile`:

```ruby
platform :ios, '8.0'
use_frameworks!
pod 'Pep'
```

To get the full benefits import `Pep` wherever you import UIKit

``` swift
